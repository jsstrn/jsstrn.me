By default, VS Code will send diagnostic data which you disable in settings. To do this, search for telemetry in settings and select off under Telemetry Level. Another approach is to install VSCodium which contains the original open sourced binaries and has telemetry disabled by default. For macOS, use Homebrew to install VSCodium brew install vscodium

Installing GPG Suite gives you the GPG Keychain which makes it easier for you to view all your keys and to store your passphrase in the keychain so that you don’t have to keep typing it in all the time.
