Learn to use VS Code more efficiently

Multi-cursor editing Select your text, hit Shift-Command-L to select all occurences of the selection. Command-D to select them one at a time. Add a new cursor above or below, Option-Command + up/down arrow keys. Hit Escape key to exit multi-cursor mode. IntelliSense The default shortcut key to trigger suggestions is Control-Space or Fn-Control-Space. If this does not work, try the Option-Escape keys. Navigation in VS Code

Navigation Command-P to quickly search and open a file. Shift-Command-E to move your cursor to the Explorer pane then navigate with arrow keys, or start typing to search for a file. Control-- to navigate to a previously opened file Shift-Control-- to navigate forward. Breadcrumbs You can navigate anywhere within a file by clicking on the breadcrumbs. You can also open other files directly from the breadcrumbs. Reference Code Navigation in VS Code
