Install VSCodium instead of VS Code

By default, VS Code will send diagnostic data which you disable in settings. To do this, search for telemetry in settings and select off under Telemetry Level. Another approach is to install VSCodium which contains the original open sourced binaries and has telemetry disabled by default. For macOS, use Homebrew to install VSCodium brew install vscodium

Learn to use VS Code more efficiently

Multi-cursor editing Select your text, hit Shift-Command-L to select all occurences of the selection. Command-D to select them one at a time. Add a new cursor above or below, Option-Command + up/down arrow keys. Hit Escape key to exit multi-cursor mode. IntelliSense The default shortcut key to trigger suggestions is Control-Space or Fn-Control-Space. If this does not work, try the Option-Escape keys. Limit number of open editors in VS Code

We can limit the number of open editors in VS Code by configuring the following in settings: workbench.editor.limit.enabled: true – enable editor limit; default is false workbench.editor.limit.value: 1 – set number of open editors; default is 10 workbench.editor.limit.perEditorGroup: true – allow side-by-side panels; default is false
